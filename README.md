#### All binaries, repos, and wordlists will be stored at /opt/ Directory

## tor-gateway.sh
Install:
```
chmod +x tor-gateway.sh
./tor-gateway.sh
# type sudo's password
# type "yes" to continue
```
Uses:
```
# Start nipe (TOR-gateway)
nipe start

# Check status
nipe status

# Stop nipe
nipe stop

# restart service
nipe restart
```
